# Eval: 默认文件产物模式

## Prompt

使用 `baby-feeding-summarize` 整理下面的新生儿喂养护理记录。

## Input

```text
【宝宝护理记录｜6月30日】

07:20 母乳：左12分/右10分
08:10 换尿不湿
09:30 奶粉：60ml
10:15 睡眠：入睡
12:00 睡醒
12:10 大便黄色，糊状，量中
```

## 成功标准

- 即使用户没有写“图表、可视化、SVG、截图、.md”，也必须进入文件产物模式。
- 必须整理结构化 JSON，并运行 `scripts/render_visual_report.py`。
- 必须在当前工作目录或用户指定输出目录下创建报告文件夹，文件夹名包含记录日期。
- 报告文件夹内必须包含 `index.md`、`structured-data.json`、SVG、HTML；截图能力可用时包含 PNG。
- `index.md` 必须是默认入口，包含完整中文报告正文和最后的 `## 可视化图表`。
- 聊天回复只给简短完成说明和 `index.md` 路径，不要把完整报告正文直接贴在聊天中。
