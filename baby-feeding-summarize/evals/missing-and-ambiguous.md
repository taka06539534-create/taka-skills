# Eval: 缺失与歧义记录

## Prompt

使用 `baby-feeding-summarize` 整理下面记录。注意不要编造缺失数据。

## Input

```text
【宝宝护理记录｜6月28日】

06:40 母乳：左—分/右8分
08:00 瓶喂：45ml
09:10 换尿不湿：小便无，大便有
09:12 大便：颜色黄，性状—，量少
10:30 睡眠：入睡，睡姿侧卧，头偏右
13:00 奶粉：—ml
15:20 异常：吐奶一次，精神可
```

## 成功标准

- 左侧亲喂时长标为未记录，只统计右侧 8 分钟。
- `瓶喂：45ml` 标为瓶喂类型未明确，不归入瓶喂母乳或奶粉。
- `奶粉：—ml` 记录为一次奶粉事件，但不计入奶粉总量。
- 小便不计数，大便计 1 次，性状为未记录。
- 睡眠只确认 10:30 入睡，不能计算总睡眠时长。
- 对“吐奶一次，精神可”只提示继续观察；不下诊断。
- 参考区间对比应把缺失数据标为“无法判断/记录不足”，不能硬判定异常。
