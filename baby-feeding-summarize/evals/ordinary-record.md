# Eval: 普通完整记录

## Prompt

使用 `baby-feeding-summarize` 分析下面的新生儿护理记录，并输出中文总结。

## Input

```text
【宝宝护理记录｜6月27日】

07:20 母乳：左12分/右10分
07:45 拍嗝：8分
08:10 换尿不湿：小便有，大便无
09:30 奶粉：60ml
10:15 睡眠：入睡，睡姿仰卧，头偏左
12:00 睡眠：睡醒
12:10 瓶喂母乳：50ml
12:35 换尿不湿：小便有，大便有
12:36 大便：颜色黄色，性状糊状，量中
15:00 母乳：左8分/右9分
16:30 奶粉：70ml
18:00 体温：36.8℃
20:10 异常：无
```

## 成功标准

- 输出以“今日概览”开头，并包含喂养分析、排便排尿分析、睡眠推测、观察优先级、信息缺口；统计视图放在文档最后。
- 亲喂统计为 2 次，总时长 39 分钟，且不换算 ml。
- 奶粉统计为 2 次，总 130 ml；瓶喂母乳统计为 1 次，总 50 ml。
- 小便为 2 次，大便为 1 次；同一时间的大便事件和质量描述不重复计数。
- 喂养分析需要判断当前状态：哪些较好、哪些需要关注，并给出优化建议。
- 睡眠可推测为 10:15-12:00，约 1 小时 45 分。
- 异常写为未记录异常或无异常，不制造风险。
