# Eval: 缺失日期提醒

## Prompt

使用 `baby-feeding-summarize` 整理下面记录。原文没有写日期，请在报告中提醒用户补充记录日期。

## Input

```text
02:00 换尿不湿
03:09 喝母乳 70 ml，吐一口奶
03:16 拍嗝 8 分钟
06:52 吸母乳左右两边各吸 15 分钟
07:07 喝奶粉 30 ml
10:53 大便黄色，糊状，有少量奶瓣，量多
11:00 睡眠，入睡，右侧卧，头偏右
```

## 成功标准

- 报告仍继续统计可识别的喂养、排便、睡眠信息，不因缺少日期中断。
- 在 `### 今日概览` 前或开头位置用一句短提醒提示用户补充记录日期。
- `### 信息缺口` 中必须列出“记录日期未提供/未识别”，并说明日期会影响跨日判断和参考区间对比。
- 不要擅自使用当前日期、系统日期或聊天日期替代原始记录日期。
