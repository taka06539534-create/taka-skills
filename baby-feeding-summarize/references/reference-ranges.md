# 常见参考区间

这些区间只用于帮助家长理解趋势，不能替代医生判断。输出时必须写明：需结合宝宝日龄、胎龄、体重增长、喂养方式、精神状态、尿便变化和儿科医生建议。

## 对比口径

| 指标 | 常见参考 | 输出时怎么写 |
|---|---|---|
| 亲喂频次 | 母乳喂养新生儿常见约每 2 小时一次，约 10-12 次/24 小时 | 与记录中的亲喂+补喂节律对比；亲喂不换算 ml |
| 瓶喂频次 | 瓶喂新生儿常见每 2-3 小时一次，通常建议至少约 8 次/24 小时 | 统计瓶喂母乳和奶粉次数；若亲喂混合喂养，不简单判定偏少 |
| 瓶喂单次量 | 出生最初几天量较小，之后常见 1-2 oz/次；满月前后可到 3-4 oz/次 | 可换算为约 30-60 ml/次、90-120 ml/次；必须结合日龄 |
| 湿尿布 | 出生前几天约 2-3 次/日；出生 4-5 天后通常至少 5-6 次/日 | 按本项目口径，未明确写大便有的换尿布相关记录计作小便；明确写大便有时，小便只在原文有尿便线索时计入 |
| 大便 | 差异比小便大；约第 4 天起可见多次大便，AAP 给出至少 4 次/日的参考 | 结合颜色、性状、量和日龄；不要仅凭次数诊断 |
| 睡眠总时长 | 0-3 月常见约 14-17 小时/日，含白天小睡 | 没有睡醒时间时写“无法与参考时长对比” |
| 睡姿安全 | NICHD 建议婴儿每次睡眠从仰卧开始；睡眠表面应坚实、平坦、水平 | 记录有侧卧/趴睡时温和提示“可优化为仰卧入睡” |

## 对比措辞

- 用“相近 / 记录不足 / 可能偏少 / 可能偏多 / 需要结合日龄判断”，不要用“正常 / 不正常”。
- 对偏离参考区间的情况，先检查记录是否缺失，再给观察建议。
- 只有出现明显风险信号时才建议就医；普通波动建议“继续记录并观察”。
- 引用参考区间时不要输出外部来源长链接，除非用户要求来源。
