# Goal 模式循环协议

## 目标

在 Codex Goal 模式下，循环执行 moonlight-girl-companion skill，通过多轮选题、生成、验收和修订，产出达到公众号发布标准的午夜情感故事发布包。发布包必须包含完整文章、公众号标题、描述、真实封面图、真实章节插图、Prompt、验收报告和迭代日志；不能只有一个 Markdown 文档。

## 每轮循环步骤

1. 读取用户需求。
2. 判断工作模式：指定选题模式或自主选题模式。
3. 如果是自主选题模式，先执行选题流程。
4. 读取上一轮输出和验收报告。
5. 判断上一轮是否通过 Done Criteria。
6. 如果已经通过，停止循环并输出 PASS。
7. 如果未通过，提取失败项。
8. 判断失败项所属层级。
9. 按优先级制定修订计划。
10. 执行修订。
11. 生成或补齐真实封面图和章节插图文件。
12. 组装或更新发布包目录。
13. 运行校验脚本。
14. 输出新的质量验收报告。
15. 更新迭代日志。
16. 判断是否继续下一轮。

## 发布包组装步骤

每轮进入验收前必须检查发布包目录：

1. 写入 `article.md` 或 `index.md`，包含完整正文、标题、描述、章节插图引用、文末深夜寄语、互动话题、更多文章入口和最终状态摘要。
2. 写入 `metadata.json`，至少包含 `title`、`description`、`cover_image`、`article_file`、`illustrations`。
3. 写入真实封面图片文件，默认 `cover.png`。
4. 写入真实章节插图文件，默认 `images/chapter-01.png` 到 `images/chapter-05.png`。
5. 写入 Prompt 文件，默认 `prompts/cover_prompt.md` 与 `prompts/illustration_prompts.md`。
6. 写入 `review_report.md` 和 `iteration_log.md`。
7. 对整个发布包目录运行 `scripts/validate_article_package.py`。

## 并行插图生成

角色视觉设定卡、统一美术风格和统一色彩系统确定后，封面图和各章节插图可以并行生成，以提升执行效率。

- Codex 环境允许 subagent 时，可以为封面和每个章节图分别分派独立 subagent。
- 每个 subagent 只负责一个图片文件和对应 Prompt 草稿，例如 `cover.png` 或 `images/chapter-03.png`。
- subagent 不得修改正文、故事设定、角色视觉设定、metadata 或质量验收结论。
- 主 Agent 必须回收所有图片后统一检查人物一致性、色彩系统、章节顺序、文件路径和 Markdown 图片引用。
- 任一图片生成失败时，不得输出 PASS；应修复失败图片或输出 FAIL 并说明人工介入位置。

## 修订优先级

第一优先级：合规安全问题
第二优先级：发布包缺失问题，包括 metadata、真实封面图、真实章节图和正文图片引用
第三优先级：选题问题
第四优先级：结构缺失问题
第五优先级：字数和章节问题
第六优先级：女性视角和人物逻辑问题
第七优先级：角色视觉设定问题
第八优先级：插图一致性问题
第九优先级：情绪推进和故事节奏问题
第十优先级：文末深夜寄语、互动话题、更多文章入口缺失、顺序错误或文案生硬问题
第十一优先级：标题、摘要、语言润色问题

## 停止条件

满足以下任意条件时停止：

1. Done Criteria 全部满足，输出 PASS。
2. 达到最大迭代轮数，输出 FAIL。
3. 出现无法自动修复的问题，输出 FAIL 并请求人工介入。
4. 用户明确要求停止。

## 禁止行为

1. 不允许未通过验收就输出 PASS。
2. 不允许只给建议不执行修订。
3. 不允许每轮重复生成全新故事，除非结构严重失败或选题不成立。
4. 不允许修订后丢失用户原始核心要求。
5. 不允许修订后导致角色视觉设定变化。
6. 不允许忽略上一轮验收报告。
7. 不允许为了凑字数添加无意义段落。
8. 不允许为了情绪刺激制造低俗或猎奇内容。
9. 不允许选题不合格时继续润色正文。
10. 不允许自主选题模式下跳过候选选题评分。
11. 不允许只有 Markdown 文档、Prompt 或插图占位时输出 PASS。
12. 不允许缺少真实封面图或章节插图文件时标记为可直接发布。
13. 不允许正文结尾缺少深夜寄语、互动话题或更多文章入口时输出 PASS。
14. 不允许正文结尾的深夜寄语、互动话题、更多文章入口顺序错误时输出 PASS。
15. 不允许用恐吓、低俗、夸大、诱导隐私暴露或硬广式文案作为文末三个章节内容。
